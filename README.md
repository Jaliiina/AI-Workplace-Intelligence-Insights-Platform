# AI 职场智能洞察平台
# 项目概述
AI 职场智能洞察平台是一个聚焦于 AI 领域招聘市场的数据分析与可视化系统，通过直观的图表展示和多维度数据拆解，帮助用户了解 AI 岗位的市场分布、技能需求及薪资水平等关键信息，为职业发展和招聘决策提供数据支持。

# 项目结构
```plaintext
ai_dashboard/
    ├── app.py               # 后端路由配置
    ├── static/
    │   └── data/
    │       ├── wordcloud.json   # 技能关键词及频率数据
    │       ├── skill_drill.json # 技能维度详细数据
    │       └── city_drill.json  # 城市维度详细数据
    └── templates/
        ├── city_detail.html    # 城市详情页模板
        └── city_region.html    # 城市区域分析页模板
```

# 核心功能模块

## 1. 城市详情分析
- 通过 `/city_detail` 路由访问城市详情页面
- 展示特定城市的 AI 岗位分布、薪资水平及技能需求
- 支持城市间数据对比与趋势分析

## 2. 城市区域分布可视化
- 基于 ECharts 实现全国城市 Geo 地图可视化
- 以气泡图形式展示各城市岗位数量（气泡大小）与薪资水平
- 支持地图缩放、漫游及鼠标悬停查看详情功能
- 视觉设计：采用橙色系为主色调（#ff8c42），突出数据亮点

## 3. 技能需求分析
- 技能词云展示：通过 `wordcloud.json` 数据生成热门技能词云，包含 Spring、Linux 系统、数据结构与算法等高频技能
- 技能深度拆解：`skill_drill.json` 提供各技能对应的岗位数量、薪资分布（箱线图数据）、经验与薪资关系及城市分布
- 核心技能分类：涵盖人工智能、机器学习、深度学习、大数据处理等关键领域


# 技术实现
- 后端框架：Flask（通过 `app.py` 中的路由配置可知）
- 前端可视化：ECharts（用于地图、气泡图等数据可视化）
- 数据格式：JSON（用于存储和传递各类分析数据）
- 模板引擎：Jinja2（用于渲染 HTML 页面）

# 使用指南
1. 启动应用后，通过路由访问各功能页面
2. 城市区域地图支持交互操作： 
    - 鼠标滚轮缩放地图
    - 拖拽实现地图漫游
    - 悬停在气泡上可查看城市岗位数及平均薪资
3. 技能分析模块可查看不同技能的市场需求与薪资水平关联关系

# 数据示例
- 热门技能示例：Spring（68）、Linux 系统（68）、数据结构与算法（67）
- 核心技能分类数据：人工智能（32）、机器学习（19）、深度学习（10）
- 主要城市：北京、上海、广州、深圳、杭州等全国多城市覆盖
